# New York Times movie reviews

Application for search New York Times movie reviews and get movie critics using Javascript, React, Redux, React hooks, redux-saga and styled-components.

The application allows you search New York Times movie reviews by keyword and filter by Criticâ€™s Pick and Reviewer, also supports ordering results by title, by publication date, or by opening date. It's possible to get all New York Times movie critics, and ou can either specify the reviewer name. The movie reviews can be added to a favorites list that is stored in the application state.

The Movie Reviews API from [The New York Times Developer Network](https://developer.nytimes.com/) was used to get all the data.
